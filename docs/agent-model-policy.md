# מדיניות ניהול סוכנים ומודלים — לולאת הביקורת העיצובית

האורקסטרטור הראשי (Claude Fable 5, הסשן הראשי) מנהל את הלולאה ומחליט לכל
משימה איזה סוכן ואיזה מודל מריצים אותה. העיקרון: המודל החזק ביותר רק היכן
ששיפוט עדין מכריע; מודלים זולים ומהירים לכל מה שמכני.

## חלוקת תפקידים

| תפקיד | מודל | רציונל |
|---|---|---|
| אורקסטרטור: טריאז' ממצאים, החלטות ארכיטקטוניות, תיקונים חוצי-קומפוננטות, ניהול הלולאה | **Fable** (הסשן הראשי) | ההחלטות היקרות ביותר — מה מיישמים, מה דוחים, מתי עוצרים |
| מבקר עיצוב ("קלוד דיזיין") — קריאת צילומים, שיפוט מול רף Linear/Vercel/Arc | **Opus** | שיפוט ויזואלי חזק; אין צורך ב־Fable כשהפורמט והרף מוגדרים היטב |
| מיישם — תיקוני CSS/JSX תחומים מתוך רשימת תיקונים מפורשת | **Sonnet** | עבודה ממוקדת עם קריטריון הצלחה ברור; מהיר וזול |
| מאמת — הרצת lint/typecheck/build/Playwright, צילומי מסך, דוח עבר/נכשל | **Haiku** | מכני לחלוטין; התוצאה בינארית |

הערות:
- תיקון שנוגע ל־state, לוגיקת observer, או שינוי חוזה בדיקות — נשאר אצל
  האורקסטרטור (Fable), לא יורד ל־Sonnet.
- שרשרת אימות קצרה (פקודה-שתיים) האורקסטרטור מריץ ישירות — ספאון של סוכן
  יקר מהרצה ישירה. סוכן Haiku נכנס רק כשמדובר בסבב אימות מלא + צילומים
  שאפשר להריץ במקביל לעבודה אחרת.

## רציפות בין סבבים

הקשר של המבקר לא תלוי ב־transcript של סוכן יחיד: כל סבב נרשם
ב־`docs/design-review-rounds.md` (ממצאים ← מה יושם ← מה נדחה ולמה).
כשמאתחלים מבקר חדש (או מחליפים מודל), נותנים לו את הקובץ הזה — כך הוא
לא חוזר על פידבק שכבר טופל, בלי תלות בזיכרון סוכן.

## חוזה הלולאה (איטרציה אחת)

1. **צילום** — `node scripts/capture-review-shots.mjs rN` מול production
   build על :3101 (לעולם לא dev — ה־overlay מלכלך את הפריים).
2. **ביקורת** — סוכן Opus מקבל את הצילומים + `design-review-rounds.md`,
   מחזיר VERDICT / TOP FIXES / MOBILE / POLISH.
3. **טריאז'** — האורקסטרטור ממיין: יישום עצמי / Sonnet / דחייה מנומקת.
4. **יישום** — לפי הטבלה למעלה.
5. **אימות** — lint, typecheck, build, מלוא בדיקות ה־Playwright. אדום =
   מתקנים לפני שממשיכים.
6. **תיעוד וקומיט** — עדכון `design-review-rounds.md`, קומיט אחד לסבב.
7. **עצירה** — כשהמבקר מדווח התכנסות (אין ממצאים מעל רמת POLISH) או
   כשהאורקסטרטור מזהה תשואה פוחתת, הלולאה נעצרת ומדווחים למשתמשת.
